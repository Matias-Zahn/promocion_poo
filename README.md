# PROYECTO NOC_APP

Se desarrolló una aplicación "NOC" (Network Operations Center). El objetivo de esta aplicación es monitorear un servicio web en intervalos regulares y persistir un log con el resultado de dicho chequeo.


### PASOS PARA INICIALIZAR DICHA APP

1. Ejecutar el script `npm install` para realizar las instalaciones requeridas.
2. Ejecutar el script `npm run dev` para comenzar a correr el proyecto
